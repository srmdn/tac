# Getting Started

## Installation

Install Tac via npm:

```bash
npm install -g @srmdn/tac
```

Or use npx (no install required):

```bash
npx @srmdn/tac init my-project
```

## Initialize a Project

Create a new Tac project:

```bash
tac init my-project
cd my-project
```

This scaffolds a minimal project structure with sensible defaults.

## Running Tac

Start the development server:

```bash
tac dev
```

Build for production:

```bash
tac build
```
