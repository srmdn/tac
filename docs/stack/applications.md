# Applications

The user-facing layer of the Agent Stack.

## Categories

### Coding Agents

Autonomous code generation, refactoring, and debugging. Examples: Claude Code, OpenClaw, GitHub Copilot Workspace.

### Personal AI Assistants

General-purpose agents with long-term memory and multi-modal capabilities. Examples: Kimi, custom GPTs.

### Support Bots

Domain-specific agents trained on knowledge bases. Handle tier-1 support, escalation triage.

### Research Agents

Autonomous information gathering, synthesis, and reporting. Browse the web, read papers, compile summaries.

## What Separates a Demo from a Product

| Demo | Product |
|------|---------|
| Single-turn | Multi-turn with memory |
| One model | Routed, cached, observed |
| Hardcoded tools | MCP-discoverable tools |
| Happy path | Error handling, retries, fallbacks |
| Local run | Deployed, scalable, monitored |

The gap is the stack below the surface.
